# Claude Code Prompt: Build a Contractor Website Inspired by Stirling Interiors

## Reference Site

Use this site as the main design inspiration:

https://stirlinginteriors.com/

Do **not** copy their text, images, logo, code, brand, or exact layout. Use it as inspiration for structure, pacing, tone, and overall website quality.

The reference site is for an office design, fit-out, refurbishment, and furniture company. The useful design ideas to borrow are:

- A polished service-business feel
- A strong hero section with a clear positioning statement
- A professional navigation structure
- A services-heavy homepage
- Trust-building sections
- Client/logo proof
- Testimonials
- Portfolio/work examples
- Strong local/service-area messaging
- A clean footer with contact details
- A tone that feels established, capable, and reliable

Build a similar quality website, but adapt it for a **general contractor / local service business**.

---

# Project Goal

Create a modern, professional, mobile-first website for a local general contractor.

The site should help the business look more established and trustworthy online, especially compared to competitors that only use Facebook or have outdated websites.

The website should feel:

- Clean
- Professional
- Local
- Trustworthy
- Strong
- Established
- Easy to navigate
- Simple for customers to contact

Avoid making it look too flashy, overly futuristic, or like a generic SaaS landing page. This is for a local service business, so it should feel grounded and practical.

---

# Overall Website Structure

Build a one-page website with clear sections.

Recommended sections:

1. Header / navigation
2. Hero section
3. Trust bar
4. About section
5. Services section
6. Featured work / gallery
7. Why choose us
8. Testimonials
9. Process section
10. Final call-to-action
11. Contact section
12. Footer

---

# Header / Navigation

Create a clean sticky or fixed header.

Include:

- Business name or logo placeholder
- Navigation links:
  - Home
  - Services
  - Work
  - About
  - Contact
- Primary CTA button:
  - Request a Quote
  - Get an Estimate
  - Call Now

The header should feel similar in professionalism to the Stirling Interiors site, but adapted for a contractor.

Use strong spacing, simple typography, and a clean layout.

On mobile, use a simple hamburger menu or compact stacked layout.

---

# Hero Section

The hero is the most important part.

It should immediately explain:

- What the company does
- Where they work
- Why they can be trusted
- How to contact them

Suggested headline style:

**Reliable Contracting Services Built Around Quality Work**

Alternative headline options:

- Quality Renovations & Exterior Work Across HRM
- Local Contractors You Can Count On
- Built Right. Finished Properly.
- Professional Contracting Services for Homes and Businesses

Supporting text example:

> Serving Halifax, Dartmouth, Cole Harbour, and surrounding areas with reliable contracting, renovation, and property improvement services.

Hero CTA buttons:

- Request a Free Quote
- View Our Work

Hero visual:

Use a strong contractor-style image placeholder, gallery image, or background image. If real images are available in the project, use them. If not, use a clean placeholder area that can later be swapped with real jobsite/project photos.

Design notes:

- Use a large headline
- Keep body text short
- Use a strong CTA
- Add a small trust note like:
  - Local service business
  - Free estimates
  - Fully insured
  - Residential and commercial work

---

# Trust Bar

Below the hero, add a simple trust bar.

Examples:

- Fully Insured
- Free Estimates
- Local Service Area
- Residential & Commercial
- Quality Workmanship

This should be visually simple, maybe with icons or small cards.

Purpose:
Give visitors immediate confidence before they scroll.

# About Section

Inspired by the Stirling Interiors homepage, include a short about section that explains the business in a human and professional way.

Do not write a huge wall of text.

Example copy:

> We help homeowners and businesses complete projects with clear communication, reliable workmanship, and attention to detail from start to finish.

Add another short paragraph:

> Whether it is a renovation, exterior upgrade, repair, or larger property improvement project, our goal is to make the process straightforward and the finished work something you are proud of.

Design idea:

- Two-column layout
- Text on one side
- Image/card/stats on the other

Possible stats:

- Local service
- Free estimates
- Residential and commercial
- Quality-first approach

# Featured Work / Gallery

This section is very important.

Contractor websites need proof.

Create a clean project gallery with image cards.

Each card can include:

- Project type
- Short caption
- Location placeholder
- Before/after label if useful

Example:

- Exterior Upgrade
- Deck Build
- Roof Repair
- Interior Renovation
- Property Maintenance Project
- Commercial Repair

If no real photos are available, use placeholders but make the layout ready for real images.

Design direction:

- Large image cards
- Clean spacing
- Slight rounded corners
- Minimal text overlay

# Testimonials Section

Add 2–3 testimonial cards.

Use placeholder testimonials if real ones are not available.

Example:

> Great communication, clean work, and the finished result was exactly what we were hoping for.

> Professional from start to finish. They showed up when they said they would and did quality work.

> The project was handled well and the end result made a big difference to the property.

---

# Process Section

The Stirling Interiors site emphasizes a professional process. Add a simplified version for a local contractor.

Use 3 or 4 steps:

1. Get in Touch
2. Discuss the Project
3. Receive a Quote
4. Complete the Work

Example:

## 1. Get in Touch
Send a message or call with a few details about the work you need done.

## 2. Review the Project
We discuss the job, timeline, materials, and what matters most to you.

## 3. Get a Clear Quote
You receive a straightforward estimate with the scope of work clearly laid out.

## 4. Work Gets Done
The project is completed with care, communication, and attention to the final result.

# Visual Design Direction

Use a design system that feels premium but still appropriate for a local contractor.

## Colors

Recommended palette:

- Off-white / warm white background
- Charcoal or dark navy text
- Light grey sections
- One strong accent color

Accent color options:

- Muted blue
- Deep green
- Burnt orange
- Gold/tan
- Red if the brand already uses it

Do not overuse the accent color. Use it mainly for buttons, highlights, icons, and small details.

## Typography

Use clean, modern sans-serif fonts.

Good options:

- Inter
- Sora
- Montserrat
- Poppins
- Manrope

Headlines should be bold and confident.
Body text should be readable and not too small.

## Spacing

Use generous spacing.
Avoid cramped sections.

The site should feel calm, clear, and professional.

## Cards

Use soft shadows or thin borders.
Use rounded corners, but not overly playful.

## Images

Use real project images if available.
If images are placeholders, make the layout ready for real job photos.

Contractor websites depend heavily on real work photos.

Optional:
Add a sticky mobile CTA button:

---

# Content Tone

The writing should be:

- Clear
- Local
- Trustworthy
- Human
- Confident
- Simple

Avoid:

- Corporate buzzwords
- Overly dramatic language
- Generic marketing fluff
- Claims that cannot be verified
- Fake guarantees

Good tone example:

> We focus on reliable work, clear communication, and finished projects that hold up.

Bad tone example:

> We deliver next-generation property transformation solutions optimized for modern homeowners.

---

# Important Implementation Notes

- Build the site cleanly and professionally.
- Keep code organized.
- Use reusable sections/components if this is a React/Next.js project.
- Make it easy to swap business name, photos, services, phone number, and location.
- Keep the design polished enough to use as a portfolio example.
- Do not copy Stirling Interiors directly.
- Use Stirling Interiors as inspiration for the type of service-business structure and polished trust-building layout.

Create a complete one-page contractor website that feels like a professional local service business site inspired by the structure and quality of Stirling Interiors.
